프로젝트 설명<br>
게시판의 간단한 기능인 CRUD를 이용<br>
<br>

입력<br>
데이터 입력 기능<br>
출력<br>
데이터 출력과 선택 출력 기능 추가. <br>
수정<br>
간단한 수정<br>
삭제<br>
데이터 삭제 기능.<br>
<br>

부가적인 기능 설명:<br>
<br>
자바스크립트<br>
jsp 파일내 간단한 자바 스크립트를 추가하여 받은 객체의 정보를 직접적으로 클릭 했을 때에 이벤트페이지가 호출되도록 기능 추가.<br>
코드내 여러 선택자를 이용하여 다양한 방법으로 같은 기능을 구현.<br>

실행방법.<br>
<br>
이클립스내 깃헙기능 사용및 Project Import.<br>
아파치 톰캣9 버전 기본 라이브러리 추가.<br>
아파치 톰캣 내 context 경로를 기본 '/' 으로 변경 및 설정.<br>
실행 환경 설정 후 서버 실행.<br>
